My work sits at the intersection of data systems, predictive modeling, and user-facing products, from clinical-trial metadata APIs to construction cost forecasting and interactive prediction games.

- Currently building predictive S-curve cost projection tools with UCSD Capital Program Management

## Featured Projects

### UCSD Capital Program Management S-Curve Forecasting

Building a predictive model that forecasts cumulative construction project cost curves from square footage, estimated cost, project duration, and planning metrics. The tool supports leadership-facing analytics through a clean Vercel page and draws on 14+ years of construction project data.

### Spotify Popularity Prediction Game

Created an interactive web game using 10,000+ Spotify tracks where users predict song popularity and compare their guesses against real-world audio feature trends. Built with Python, Pandas, HTML, CSS, and JavaScript.

### Trash Classifier

Collaborated with an ACM Projects team to train a PyTorch convolutional neural network that classifies 7 waste categories, then deployed the model behind a Flask and React web app for real-time image upload and inference.

### Clinical Trial Metadata Pipelines

Integrated Hugging Face transformer models and FastAPI endpoints into data workflows that extract and validate structured clinical trial metadata from unstructured protocol documents.
